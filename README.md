# de-workbench

A collection of Python CLI tools for data engineers.

## Tools

- **sql_formatter** — Format messy SQL into clean readable SQL
- **schema_compare** — Compare two table schemas and show differences  
- **data_profiler** — Quick statistical profile of any CSV or Parquet file
- **query_timer** — Benchmark SQL query execution time
- **manifest_reader** — Read dbt manifest.json and print model summary

## Installation
```bash
pip install de-workbench
```
